# Документация EasyBooker

## В этом репозитории

- `README.md` — обзор, лицензия, авторство, рабочий бот в Telegram.
- `CONTRIBUTING.md` — как участвовать (PR, issues).
- `SECURITY.md` — ответственное раскрытие уязвимостей.
- `docs/discovery/` — **материалы discovery** (сценарии, SWOT, рынок робомоек, заметки по презентации, фрагмент ТЗ) в **обезличенной** редакции.
- `docs/SETUP.md` — запуск локально и в продакшене (базовая схема).
- `docs/ATTRIBUTION.md` — связь с исходным репозиторием разработки.
